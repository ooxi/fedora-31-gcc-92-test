# Fedora 31 GCC 9.2 Test

Since I have trouble getting [GCC 9.2][gcc-92] to work, this repository contains
a simple example (blinking led) from [Advanced Software Framework][asf],
comparing [Fedora 30](Dockerfile.f30) with [Fedora 31](Dockerfile.f31).

[gcc-92]: https://apps.fedoraproject.org/packages/arm-none-eabi-gcc-cs
[asf]: https://www.microchip.com/mplab/avr-support/advanced-software-framework


## Important files

* [C source code](asf/sam0/applications/led_toggle/led_toggle.c)
* [Makefile](asf/sam0/applications/led_toggle/samd21_xplained_pro/gcc/config.mk)
* [Linker script](asf/sam0/utils/linker_scripts/samd21/gcc/samd21j18a_flash.ld)


## Quickstart using host

```
make -C 'asf/sam0/applications/led_toggle/samd21_xplained_pro/gcc'
```


## Quickstart using Docker

```
docker build -f 'Dockerfile.f30' -t 'test-f30' .
docker build -f 'Dockerfile.f31' -t 'test-f31' .

docker run -it --rm -v $(shell pwd):/src 'test-f30' make -C '/src/asf/sam0/applications/led_toggle/samd21_xplained_pro/gcc'
docker run -it --rm -v $(shell pwd):/src 'test-f31' make -C '/src/asf/sam0/applications/led_toggle/samd21_xplained_pro/gcc'
```
